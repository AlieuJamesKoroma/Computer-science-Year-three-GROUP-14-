# Computer-science-Year-three-GROUP-14-

Android-Based Attendance Management System

Overview

The Android-Based Attendance Management System is a mobile application developed using React Native, Expo, and Visual Studio Code. The application is designed to help lecturers, teachers, and institutions manage student attendance digitally, eliminating the need for paper-based attendance records.

This project is being developed as a learning project and will be expanded with additional features over time.

Features

- Student registration
- View student list
- Mark attendance (Present/Absent)
- Attendance history
- Search students
- Local data storage
- User-friendly interface

Planned Features

- Lecturer login and authentication
- Firebase cloud database
- QR Code attendance
- Student photo registration
- Fingerprint/Biometric attendance
- PDF attendance reports
- Excel export
- Dark mode
- Push notifications


Technologies Used

- React Native
- Expo SDK 56
- JavaScript (ES6+)
- Visual Studio Code
- Expo Go
- React Navigation
- AsyncStorage

Installation

Clone the repository:

git clone https://github.com/alieujameskoroma367144@gmail.com/AttendanceSystem.git

Open the project:

cd AttendanceSystem

Install dependencies:

npm install

Start the Expo development server:

npx expo start

Scan the QR code using the Expo Go app on your Android device.
